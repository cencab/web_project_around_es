# Tripleten web_project_around_es

**🔗 Ver el proyecto en vivo:** https://cencab.github.io/web_project_around_es/

## Descripción del Proyecto

Este proyecto es una página interactiva donde los usuarios pueden compartir fotografías de lugares interesantes. Permite gestionar un perfil personal y administrar una galería de tarjetas visuales de manera dinámica. El objetivo principal fue transicionar de un sitio estático a una aplicación funcional utilizando JavaScript para manipular el DOM.

## Funcionalidades

El sitio cuenta con las siguientes capacidades interactivas:
**Edición de Perfil:** Los usuarios pueden actualizar su nombre y descripción profesional a través de un modal que se sincroniza con la información actual de la página.
**Gestión de Tarjetas:** Añadir nuevas tarjetas con títulos personalizados y enlaces de imagen. Eliminar tarjetas existentes mediante un botón dedicado. Dar "Me gusta" a las fotografías mediante un botón de corazón interactivo.
**Vista Ampliada:** Al hacer clic en cualquier imagen, se abre un modal de pantalla completa para apreciar los detalles de la fotografía y su título.
**Validación de Datos:** Manejo de parámetros predeterminados para asegurar que las tarjetas siempre tengan contenido, incluso si faltan datos en el array inicial.

## Tecnologías Utilizadas

**HTML5:** Estructura semántica y uso de elementos <template> para renderizado dinámico.
**CSS3:** Diseño responsivo siguiendo la metodología BEM (Block Element Modifier).
**JavaScript (ES6):** Lógica de interacción, manipulación del DOM, manejo de eventos y programación funcional.
